# Career Constellation dot AI

## Problem Statement: The Fertility Window VS Career Acceleration Dilemma

Young women face rising pressure to optimise career early while managing long-term biological trade-offs. The system forces a  choice it never asks of men. How might we build decision-support tools that give young women realistic, empowering data about long-term life planning without fear-based narratives?

## Solution

A career projection visualisation based on data (typical career progressions, effects due to family planning).

## Description

The career projection is visualised using a 3D graph.

- Add nebulas and stars in the background
- Each graph node is a star
- Graph renders flat on the same plane as the screen as if it was 2D
- Graph nodes expand upwards like a tree
- User can use mouse click to move around the graph, but cannot change angle
- When clicking into a star, it zooms in and the graph tilts 90 degrees so that the plane of the graph is normal to the screen
- Detailed information is displayed as a text box that appears when a node is clicked
